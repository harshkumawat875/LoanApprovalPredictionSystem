# 🤝 Contributing to Loan Approval Prediction System

Thank you for your interest in contributing! This document provides guidelines for contributing to this project.

## 🚀 Getting Started

1. **Fork the repository**
2. **Clone your fork**
   ```bash
   git clone https://github.com/your-username/loan-approval-prediction.git
   ```
3. **Create a feature branch**
   ```bash
   git checkout -b feature/your-feature-name
   ```

## 💻 Development Setup

1. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

2. **Run the application**
   ```bash
   python app.py
   ```

3. **Access the application**
   Open `http://127.0.0.1:5000` in your browser

## 📝 Code Style

- Follow PEP 8 for Python code
- Use meaningful variable and function names
- Add comments for complex logic
- Keep functions small and focused

## 🧪 Testing

- Test your changes thoroughly
- Ensure the ML model still works correctly
- Verify all dashboard features function properly
- Test on different screen sizes (mobile responsiveness)

## 📋 Pull Request Process

1. **Update documentation** if needed
2. **Add your changes** to CHANGELOG.md
3. **Ensure your code** follows the style guidelines
4. **Create a pull request** with a clear description

## 🐛 Bug Reports

When reporting bugs, please include:
- Steps to reproduce
- Expected behavior
- Actual behavior
- Screenshots (if applicable)
- Browser/OS information

## 💡 Feature Requests

For new features, please:
- Check if it already exists
- Describe the use case
- Explain why it would be valuable
- Consider implementation complexity

## 📞 Questions?

Feel free to open an issue for any questions or discussions about the project.
